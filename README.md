# Booking Court

An app to:
  - Create courts
  - Book courts

## [Live Version](http://immense-journey-96355.herokuapp.com/)

## Installation

### Client

    cd client

    npm i

    npm start

### Server

    cd server

    bundle

    rails s

## Technologies used

  - React
  - Ruby on Rails
